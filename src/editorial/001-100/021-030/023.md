# No.023 チョコボール - 200
for文を使って実際に食べたチョコボールの数をシミュレートしてみましょう。<br>
食べたチョコボールの数が持っているチョコボールの数を超える日がチョコボールがなくなる日になります。<br>
### 実装例
```py
choco_ball = 20

sum = 0 # 食べたチョコボールの数を表す変数
day = 1 # 日数を数える変数

while True:
    sum += day # 食べる個数は食べる日の数と同じなのでdayを足す
    if sum >= choco_ball: # 食べたチョコボールの数が持っているチョコボールの数を超えたらループを抜ける
        break
    day += 1 # 日数を1増やす

print(day)
```
