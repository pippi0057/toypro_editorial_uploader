# No.023 チョコボール - 200
for文を使って実際に食べたチョコボールの数をシミュレートしてみましょう。<br>
食べたチョコボールの数が持っているチョコボールの数を超える日がチョコボールがなくなる日になります。<br>
### 実装例
```py
choco_ball = 20

# 食べたチョコボールの数を表す変数
sum = 0
# 日数を数える変数
day = 1

while True:
    # 食べる個数は食べる日の数と同じなのでdayを足す
    sum += day
    # 食べたチョコボールの数が持っているチョコボールの数を超えたらループを抜ける
    if sum >= choco_ball:
        break
    # 日数を1増やす
    day += 1

print(day)
```
